ระบบขายหน้าร้านสำหรับร้านขนมโตเกียว — ใช้งานได้เลยจากเบราว์เซอร์ ไม่ต้องติดตั้งโปรแกรมเพิ่ม

## 📁 ไฟล์ในโปรเจกต์

```
Pos Tokyo numchok/
├── index.html   ← หน้าขายหลัก (สำหรับพนักงาน)
├── admin.html   ← หน้าผู้ดูแล (สำหรับเจ้าของร้าน)
├── style.css    ← ไฟล์ตกแต่งหน้าตา
└── README.md    ← ไฟล์นี้
```

## 🚀 วิธีเปิดใช้งาน (สำหรับคนที่ไม่เก่งเทคนิค)

### วิธีที่ 1 — เปิดไฟล์ตรงๆ (ง่ายที่สุด)

1. เปิด **File Explorer** ไปที่โฟลเดอร์ `Pos Tokyo numchok`
2. ดับเบิลคลิกที่ไฟล์ **`index.html`**
3. เบราว์เซอร์จะเปิดขึ้นมาโดยอัตโนมัติ ✅
4. หน้าแอดมิน: ดับเบิลคลิกที่ **`admin.html`**

> ⚠️ **หมายเหตุ:** วิธีนี้ใช้ได้กับ Chrome, Edge, Firefox ทุกเวอร์ชัน

---

### วิธีที่ 2 — รันด้วย VS Code Live Server (แนะนำ)

วิธีนี้ทำให้หน้าเว็บ **รีเฟรชอัตโนมัติ** เมื่อมีการแก้ไขไฟล์

**ขั้นตอน:**

1. **ติดตั้ง VS Code** (ถ้ายังไม่มี)
   - ดาวน์โหลดฟรีที่ https://code.visualstudio.com
   - กด Download แล้วติดตั้งตามปกติ

2. **ติดตั้ง Extension "Live Server"**
   - เปิด VS Code
   - กดไอคอน Extensions ทางซ้าย (รูปกล่อง 4 ช่อง) หรือกด `Ctrl+Shift+X`
   - พิมพ์ `Live Server` ในช่องค้นหา
   - กด **Install** บน Live Server โดย Ritwick Dey

3. **เปิดโฟลเดอร์โปรเจกต์**
   - ใน VS Code กด `File → Open Folder`
   - เลือกโฟลเดอร์ `Pos Tokyo numchok`

4. **เริ่มเซิร์ฟเวอร์**
   - คลิกขวาที่ไฟล์ `index.html` ในแผงซ้าย
   - เลือก **"Open with Live Server"**
   - เบราว์เซอร์จะเปิดขึ้นที่ `http://127.0.0.1:5500/index.html`

5. **เปิดหน้า Admin**
   - เปลี่ยน URL เป็น `http://127.0.0.1:5500/admin.html`
   - หรือกดปุ่ม "⚙️ ผู้ดูแลระบบ" ในหน้าขาย

## 📱 วิธีใช้งาน — หน้าขาย (index.html)

### 1. กรอกข้อมูลลูกค้า (ถ้ามี)
- ชื่อลูกค้า, เบอร์โทร, เวลารับ — กรอกหรือไม่กรอกก็ได้

### 2. เลือกเมนู
- กดแท็บด้านบน: **ไส้คาว / ไส้หวาน / ไส้พิเศษ / ท็อปปิ้ง**
- คลิกการ์ดเมนูที่ต้องการ → หน้าต่างตัวเลือกจะเด้งขึ้นมา

### 3. ปรับตัวเลือก (ในหน้าต่างตัวเลือก)
- เลือกตัวเลือกเฉพาะเมนู (ถ้ามี เช่น ราดนม/ช็อคโกแลต)
- เพิ่มท็อปปิ้ง (กดหลายอันได้)
- เลือกซอส / พริกไทย / ซอสแม็กกี้
- ปรับจำนวน
- กด **"เพิ่มลงตะกร้า"**

### 4. ชำระเงิน
- ดูยอดรวมในตะกร้าด้านขวา
- กด **"💳 ชำระเงิน"**
- ใส่จำนวนเงินที่รับมา → ระบบคำนวณเงินทอนให้อัตโนมัติ
- กด **"✅ ยืนยันชำระเงิน"**
- ใบเสร็จจะแสดงขึ้นมา และบันทึกลงระบบ

### 5. ส่งออเดอร์ LINE
- กด **"💚 ส่งออเดอร์ผ่าน LINE"** ก่อนหรือหลังชำระเงิน
- แอป LINE จะเปิดขึ้นพร้อมข้อความออเดอร์สำเร็จรูป

## 🔧 วิธีใช้งาน — หน้าผู้ดูแล (admin.html)

### แท็บ 📊 แดชบอร์ด
- ดูยอดขายวันนี้ และยอดรวมทั้งหมด
- ดูเมนูขายดีอันดับ 1-5

### แท็บ 🍡 จัดการเมนู
- **เพิ่มเมนูใหม่:** กรอกชื่อ ราคา หมวด แล้วกด "บันทึก"
- **แก้ไขเมนู:** กดปุ่ม ✏️ แก้ไข ข้างเมนูที่ต้องการ
- **ลบเมนู:** กดปุ่ม 🗑️ ลบ (จะถามยืนยันก่อน)
- การเปลี่ยนแปลงจะมีผลทันทีในหน้าขาย

### แท็บ 📋 ประวัติการขาย
- ดูรายการขายทั้งหมด (เรียงจากล่าสุด)
- กรองตามวันที่ได้
- คลิกที่แถวเพื่อดูรายละเอียดออเดอร์
- กด "ล้างข้อมูล" เพื่อลบประวัติทั้งหมด (ระวัง!)

## 💾 การจัดเก็บข้อมูล

ข้อมูลถูกเก็บใน **localStorage ของเบราว์เซอร์** บนเครื่องนั้นๆ

| ข้อมูล | รายละเอียด |
|--------|------------|
| เมนู (`tokyo_menu`) | แก้ไขได้ผ่านหน้า Admin |
| ประวัติการขาย (`tokyo_sales`) | บันทึกทุกครั้งที่ชำระเงิน |

> ⚠️ **สำคัญ:** ถ้าลบ Cache หรือ Clear Browser Data ข้อมูลจะหาย  
> แนะนำให้สำรองข้อมูลโดยการ Export จาก Console ของเบราว์เซอร์เป็นระยะ

## ❓ คำถามที่พบบ่อย

**Q: เปิดไฟล์แล้วหน้าตาเพี้ยน / ไม่มีสีสัน**  
A: ตรวจสอบว่าไฟล์ `style.css` อยู่ในโฟลเดอร์เดียวกับ `index.html`

**Q: กด LINE แล้วไม่มีอะไรเกิดขึ้น**  
A: ต้องติดตั้งแอป LINE บนเครื่องนั้น หรือใช้บนมือถือที่มี LINE

**Q: ข้อมูลหายไปหลังปิดเบราว์เซอร์**  
A: ข้อมูลจะยังอยู่ถ้าไม่ได้ Clear Browser Data — ลองเปิดใหม่และตรวจสอบ

**Q: จะใช้งานบนมือถือได้ไหม**  
A: ได้ — ระบบรองรับหน้าจอมือถือ เปิด URL เดียวกันในมือถือ
